Experiments

This repository contains some simple learning experiments with HTML, CSS, Javascript, D3, jQuery, and other libraries. Experiments are in reverse chronological order.

  1. Calculator. Handles simple calculations, not well tested, and doesn't handle exceptions properly. An exercise in playing with buttons.
  2. Game of Life: uses jQuery and jQuery UI to add a slider plus a few other elements.
  3. Game of Life, implemented using D3.
